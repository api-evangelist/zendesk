# Zendesk

> Zendesk is a customer-service and CRM platform. Its REST APIs cover the Support
> ticketing system (tickets, users, organizations, groups, business rules,
> channels, routing, SLAs) plus the Sell CRM (accounts, contacts, deals,
> activities). This file is generated by API Evangelist from the catalog and the
> artifacts harvested in this repository; it is not published by Zendesk.

Base URL (Support): https://{subdomain}.zendesk.com/api/v2
Authentication: API token (Basic "{email}/token") or OAuth 2.0 bearer.
Docs: https://developer.zendesk.com/api-reference/ticketing/introduction/

## Core APIs

- [Tickets](https://developer.zendesk.com/api-reference/ticketing/tickets/tickets/): Create, read, update, delete support tickets.
- [Users](https://developer.zendesk.com/api-reference/ticketing/users/users/): Manage end users, agents, and admins.
- [Organizations](https://developer.zendesk.com/api-reference/ticketing/organizations/organizations/): Manage the companies users belong to.
- [Groups](https://developer.zendesk.com/api-reference/ticketing/groups/groups/): Agent groups for routing and workflows.
- [Search](https://developer.zendesk.com/api-reference/ticketing/ticket-management/search/): Query tickets, users, and organizations.
- [Macros](https://developer.zendesk.com/api-reference/ticketing/business-rules/macros/), [Triggers](https://developer.zendesk.com/api-reference/ticketing/business-rules/triggers/), [Automations](https://developer.zendesk.com/api-reference/ticketing/business-rules/automations/): Business rules.
- [SLAs](https://developer.zendesk.com/api-reference/ticketing/business-rules/sla_policies/): Service level agreement policies.
- [OAuth](https://developer.zendesk.com/api-reference/ticketing/oauth/oauth_tokens/): OAuth 2.0 tokens and scopes.
